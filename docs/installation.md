# Installation

Install Blueprint Insight into your Unreal Engine project.

---

## Steps

1. **Download** the plugin from [Fab](https://www.fab.com/) or your distribution source.
2. **Copy** the `Blueprint_Insight` folder into your project’s `Plugins` directory:
   ```
   YourProject/Plugins/Blueprint_Insight
   ```
3. **Restart** the editor (or build the project) so the plugin compiles and loads.
4. **Open** **Window → Blueprint Insight**, or right-click a Blueprint in the Content Browser and choose **Analyze with Blueprint Insight**.

---

## Troubleshooting

- **Plugin won’t load** — Ensure the plugin is in `YourProject/Plugins/Blueprint_Insight`, you’re on **Unreal Engine 5.5 or 5.6** (or compatible 5.x), and the project has been built or the editor restarted. Enable the **Data Validation** plugin if required. Check the Output Log for errors.
- **More help** — See [Support](support.md) for FAQ and contact, or the full [Documentation](documentation.md) for features and usage.
